Endervator
----------

![Endervator Image]({{ page.endervator-image }})
{: .ui .image .centered .inline style="max-width: 256px"}

### Description

This project is heavily inspired by a feature in the mod [OpenBlocks](https://github.com/OpenMods/OpenBlocks). 
In that mod, there is a block called the Elevator.
The Elevator allows the player to move up and down between blocks through teleportation. There's a creature in the game called an 
Enderman that naturally teleports as a form of locomotion.
This project harnesses the power of the Endermen to provide locomotion to the player. 
The Endervator is a mid to late game machine that requires the player to obtain items from both of the alternate dimensions of the game.

Many-Things
-----------

<!-- ![Many-Things Image]({{ page.manythings-image }})
{: .ui .image .centered .inline style="max-width: 256px"} -->

### Description

This project is a place to put little changes and additions I have come up with into the game that don't necessarily deserve their own mod.
A couple of the changes include: being able to edit a sign by right clicking, changes to fall damage based on the block landed on, 
or even a dynamite stick item that can be attached to an arrow for an exploding shot.

MultarumOre
-----------

![Multarum Image]({{ page.multarumore-image }})
{: .ui .image .centered .inline style="max-width: 256px"}

### Description

In Minecraft, there are various ores scattered throughout the landscape. Certain ores spawn at specific depth ranges or certain biomes.
This project sets out to add a new type of ore that spawns everywhere. When broken, dice are rolled to pick a random ore to replicate.
I have implemented a configuration system to customize many facets of how this ore reacts and spawns.

---



LiquidBlocks
-----------

### Description

This project was originally intended to be a new feature for a separate project, but was deemed not a good fit.
LiquidBlocks adds the ability to create a liquid form of any solid block. When you place this liquid into the world, it flows similar 
to water except with a limited range to simulate volume.
After a set amount of time, all of the liquid will "harden" into the original block. If a redstone signal is applied to any of the liquid, 
the entire mass stays liquid.
This then allows the player to creatively "cast" structures out of desired blocks to increase building speed or to fill holes/gaps.
